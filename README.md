# 📊 Queue Visualization

## Data Structures Micro Project

### 📖 Overview

Queue Visualization is an academic microproject developed to demonstrate the working of the **Queue** data structure through interactive visualization. The project illustrates queue operations using the **First-In-First-Out (FIFO)** principle, helping learners understand how data is inserted, removed, and accessed during execution.

The visualization was created using the **Debugger Visualization** extension in Visual Studio Code, providing a clear representation of queue operations.

---

## 🎯 Objectives

* Understand the Queue data structure.
* Demonstrate FIFO (First-In-First-Out) behavior.
* Visualize queue operations interactively.
* Improve understanding of fundamental data structures.

---

## ✨ Features

* Enqueue Operation
* Dequeue Operation
* Peek Front Element
* Queue Display
* Interactive Queue Visualization
* Dynamic Update of Queue Elements
